# 图像加密与解密
基于Opencv实现的像素置乱加密解密程序。主要思路是使用一个1000×1000的随机矩阵对像素进行置乱与恢复操作。</br>
加密与解密的图片可保存，经过加密，从外观上已经无法辨认图片包含的内容。</br>
